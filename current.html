For most people, learning to code cannot be rushed. Consider the Software System Developer (SSD) program for a proper coding education with enough time to actually learn the content. Study at an institution that is recognized and trusted by industry throughout Western Canada. Over 80% of SSD graduates are hired as web application developers. The SSD Web Programmer option is an eight-month, full-time web application development program offered at BCIT's the downtown Vancouver campus. SSD prepares individuals for high-demand careers in the technical areas of web application development. Train for a new career through extensive hands-on practice.
